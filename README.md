# TaskFy - To do list

TaskFy é uma lista de afazeres simples, feita com foco em Bootstrap e JavaScript


## Aprendizados

Nesse projeto eu aprimorei meu uso do Bootstrap e JavaScript. Com a conlusão desse software eu percebi que tenho muitas coisas para se aprender e fixar com o uso do JavaScript, não é atoa que os principais problemas que eu enfrentei foram na hora de listar as tarefas, mas isso não me impediu de seguir e conluir o mesmo.
